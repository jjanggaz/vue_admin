# 다국어 개발 및 유지보수 관련 참고사항

    - Big Frame
    1. 화면 : 각 Label 및 버튼명 등은 화면별로 통합하여 관리
    2. 사이드바 : [sidebar.메뉴명]으로 관리
    3. 상단바 : 상단바 메뉴명은 [pageTitles.메뉴명]로 관리
    4. 상단바 Sub 메뉴 : [tabs.화면명.sub메뉴명] 2Level로 관리

    - small Frame
    1. columns : 각 화면의 데이터 테이블(그리드)는 [columns.화면명.컬럼명] 2Level로 관리
    2. common : 자주 재사용 되는 명칭 등은 [common]으로 관리
    3. placeholder : placeholder은 재사용이 많기 때문에 [placeholder]로 통합관리
    4. messages : messages은 재사용이 많기 때문에 [messages]로 통합관리 (2Level)

    - 정리가 완벽하지 않아 alert/confirm/placeholder/tab문 등이 다른곳에 있을 수 있음
    - header는 20250717 기준, 사용하지 않으나 차후 사용할 수도 있을 것 같아서 남겨둠
